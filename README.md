# Full Stack Authentication & RBAC System

## Features
- User Registration (USER / ADMIN)
- Login with JWT Authentication
- Role-Based Access Control
- Protected APIs
- React Dashboard with role-based UI

## Tech Stack
Backend:
- Java
- Spring Boot
- Spring Security + JWT
- MySQL

Frontend:
- React + TypeScript
- Axios
- Tailwind CSS

## Setup Instructions

### Backend
cd backend  
mvn spring-boot:run  

### Frontend
cd frontend  
npm install  
npm run dev  

## API Access
- /api/public → Public
- /api/user → USER
- /api/admin → ADMIN
